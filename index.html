Интеграция в браузер

Скопируйте этот скрипт и выполните его в консоли (F12) любой вкладки, чтобы подменить координаты в реальном времени:


// Скрипт подмены Geolocation API
const mockLat = 55.7558;
const mockLng = 37.6173;
navigator.geolocation.getCurrentPosition = (success) => {
  success({
    coords: {
      latitude: mockLat,
      longitude: mockLng,
      accuracy: 5,
      altitude: 150,
      heading: 0,
      speed: 0
    },
    timestamp: Date.now()
  });
};
